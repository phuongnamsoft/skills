# Ably Chat API Reference

This section of the documentation contains the API references for Ably Chat.

## Chat SDKs 

The following API references are available:

* [JavaScript](https://ably.com/docs/chat/api/javascript/chat-client.md)
* [React](https://ably.com/docs/chat/api/react/providers.md)
* [Kotlin (JVM & Android)](https://sdk.ably.com/builds/ably/ably-chat-kotlin/main/dokka/)
* [Swift](https://sdk.ably.com/builds/ably/ably-chat-swift/main/AblyChat/documentation/ablychat/)

## REST API 

A raw [REST API](https://ably.com/docs/api/chat-rest.md) is also provided - you don't need to install any SDK to use this. This is used where an Ably Chat SDK is not available for your chosen platform, or you don't wish to use an SDK due to resource constraints.

## Further information 

In addition to the API references listed, Ably's documentation provides information on how these interfaces are used. This includes key concepts such as connections, chat rooms, messages and typing indicators. You can find that information on the following pages:

* [About Ably Chat](https://ably.com/docs/chat.md)

* [Chat SDK Setup](https://ably.com/docs/chat/setup.md)

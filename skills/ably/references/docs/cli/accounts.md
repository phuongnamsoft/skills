# Accounts

Use the `ably accounts` command group to manage your Ably accounts and configured access tokens. These commands enable you to log in, switch between accounts, view account information, and log out.

## Manage accounts 

Use the following commands to log in, list configured accounts, switch between them, view current account details, and log out:

| Command | Description |
| ------- | ----------- |
| [`ably accounts current`](https://ably.com/docs/cli/accounts/current.md) | Show the current Ably account. |
| [`ably accounts list`](https://ably.com/docs/cli/accounts/list.md) | List all locally configured Ably accounts. |
| [`ably accounts login`](https://ably.com/docs/cli/accounts/login.md) | Log in to an Ably account. |
| [`ably accounts logout`](https://ably.com/docs/cli/accounts/logout.md) | Log out from an Ably account. |
| [`ably accounts switch`](https://ably.com/docs/cli/accounts/switch.md) | Switch to a different Ably account. |

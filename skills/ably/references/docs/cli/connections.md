# Connections

Use the `ably connections` command group to test connections to Ably. These commands enable you to verify connectivity using different transport protocols.

## Test connectivity 

Use the following command to verify connectivity to Ably using different transport protocols:

| Command | Description |
| ------- | ----------- |
| [`ably connections test`](https://ably.com/docs/cli/connections/test.md) | Test connection to Ably. |

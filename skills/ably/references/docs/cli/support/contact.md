# Support contact

Use the `ably support contact` command to contact Ably for assistance. This opens the Ably contact page in your default browser.

## Synopsis

<Code>

### Shell

```
ably support contact
```
</Code>

## Examples

Open the Ably contact page:

<Code>

### Shell

```
ably support contact
```
</Code>
